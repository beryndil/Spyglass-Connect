Serviços de Terceiros

O Spyglass usa os seguintes serviços de terceiros:

  • Google AdMob — para exibição de anúncios (Política de Privacidade)
  • Firebase Analytics — para análise de uso anônima (Política de Privacidade)

O uso desses serviços está sujeito aos seus respectivos termos e políticas de privacidade. Consulte nossa Política de Privacidade para detalhes sobre como os dados são tratados.
