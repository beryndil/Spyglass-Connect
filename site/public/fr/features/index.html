Spyglass Connect

Associez votre téléphone au compagnon de bureau Spyglass Connect pour accéder aux données de votre monde Minecraft via le WiFi local. Pas de serveurs cloud, pas de comptes — tout reste sur votre réseau.

Fonctionnalité Description
Appairage QR Scannez un QR code pour vous connecter — reconnexion automatique dès que vous êtes sur le même WiFi
Visionneuse de personnage Visualisez votre équipement complet — armure, armes, outils et enchantements
Visionneuse d’inventaire Parcourez votre inventaire complet, armure, main secondaire et coffre de l’Ender
Chercheur de coffres Recherchez n’importe quel objet dans tous les conteneurs — coffres, tonneaux, boîtes de shulker, entonnoirs
Localisateur de structures Trouvez villages, temples, monuments, forteresses et bien plus avec les coordonnées et la distance
Carte aérienne Carte interactive du terrain avec marqueurs de structures et position du joueur

Entièrement chiffré (ECDH + AES-256-GCM). Reconnexion automatique via mDNS. Protocole v2 avec négociation de version — les versions incompatibles affichent un message de mise à jour clair.

Thèmes

Choisissez parmi 15 thèmes intégrés pour personnaliser l’apparence de l’application — incluant des schémas de couleurs clairs, sombres, AMOLED et inspirés de Minecraft.

Hors ligne d’abord

Toutes les données de jeu sont intégrées à l’application et fonctionnent sans connexion Internet. Lorsque vous êtes en ligne, la synchronisation en arrière-plan vérifie les mises à jour toutes les 12 heures et ne télécharge que ce qui a changé.

Synchronisation des données

Spyglass Connect se synchronise automatiquement avec le dépôt Spyglass-Data. Un manifeste suit les versions par fichier pour que seules les données modifiées soient téléchargées — maintenant l’application à jour sans téléchargements volumineux.
