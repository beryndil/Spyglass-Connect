How to Join

Get on the server in under a minute.

Java Edition

  1. Open Minecraft Java Edition (version 1.21.11)
  2. Click Multiplayer from the main menu
  3. Click Add Server
  4. Enter the server address: mc.hardknocks.university
  5. Click Done, then double-click the server to connect

Bedrock Edition

  1. Open Minecraft Bedrock Edition (Windows, mobile, or console)
  2. Go to the Servers tab
  3. Scroll down and click Add Server
  4. Server Address: mc.hardknocks.university
  5. Port: 19132
  6. Click Save, then tap the server to join

Note: The server runs version 1.21.11 (Mounts of Mayhem). Make sure your game is on the correct version before connecting. Bedrock crossplay is handled automatically — no extra setup needed.

First Time on the Server?

  • Claim your land — protect your builds early. Ask in chat or check /claim if available.
  • Explore — the world is vast. Find a spot that speaks to you.
  • Be friendly — say hello in chat. We don’t bite (the mobs might).
  • Read the rules — they’re short and reasonable.
