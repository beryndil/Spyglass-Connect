## Java Edition

<ol class="steps">
<li>Open Minecraft <strong>Java Edition</strong> (version 1.21.11)</li>
<li>Click <strong>Multiplayer</strong> from the main menu</li>
<li>Click <strong>Add Server</strong></li>
<li>Enter the server address: <code>mc.hardknocks.university</code></li>
<li>Click <strong>Done</strong>, then double-click the server to connect</li>
</ol>

## Bedrock Edition

<ol class="steps">
<li>Open Minecraft <strong>Bedrock Edition</strong> (Windows, mobile, or console)</li>
<li>Go to the <strong>Servers</strong> tab</li>
<li>Scroll down and click <strong>Add Server</strong></li>
<li>Server Address: <code>mc.hardknocks.university</code></li>
<li>Port: <code>19132</code></li>
<li>Click <strong>Save</strong>, then tap the server to join</li>
</ol>

<div class="info-box">

**Note:** The server runs version **1.21.11** (Mounts of Mayhem). Make sure your game is on the correct version before connecting. Bedrock crossplay is handled automatically — no extra setup needed.

</div>

## First Time on the Server?

- **Claim your land** — protect your builds early. Ask in chat or check `/claim` if available.
- **Explore** — the world is vast. Find a spot that speaks to you.
- **Be friendly** — say hello in chat. We don't bite (the mobs might).
- **Read the [rules](/server/rules/)** — they're short and reasonable.
