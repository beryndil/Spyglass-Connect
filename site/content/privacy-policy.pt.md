> O Spyglass Connect é um aplicativo Android gratuito com suporte a anúncios, criado por Beryndil. Esta política explica quais dados o app coleta, como são usados e as escolhas que você tem.

## 1. Informações que Coletamos

O Spyglass em si **não** coleta nenhuma informação pessoal, como nomes, endereços de e-mail ou credenciais de conta. O app não exige a criação de uma conta.

Os seguintes serviços de terceiros podem coletar dados quando habilitados:

## 2. Google AdMob

O Spyglass exibe anúncios em banner via [Google AdMob](https://support.google.com/admob/answer/6128543). O AdMob e seus parceiros de mediação podem coletar:

- Identificadores do dispositivo (ID de publicidade)
- Endereço IP (para localização aproximada e segmentação de anúncios)
- Dados de uso do app relacionados a interações com anúncios

Se você **recusar anúncios personalizados** no diálogo de consentimento ou nas Configurações, o Spyglass solicita apenas anúncios não personalizados. Anúncios não personalizados não usam o ID de publicidade para segmentação, mas ainda podem usar seu endereço IP para segmentação geográfica geral.

Para mais detalhes, consulte a [Política de Privacidade do Google](https://policies.google.com/privacy).

## 3. Firebase Analytics

Quando você consente com **Analytics**, o Spyglass usa o Firebase Analytics para coletar padrões de uso anônimos, como quais telas são visualizadas e quais funcionalidades são usadas. Esses dados ajudam a melhorar o app. Nenhuma informação pessoalmente identificável é coletada.

Você pode ativar ou desativar o analytics a qualquer momento em Configurações > Privacidade e Dados.

## 4. Firebase Crashlytics

Quando você consente com **Relatórios de Erros**, o Spyglass usa o Firebase Crashlytics para coletar automaticamente logs de falhas, stack traces e informações do dispositivo (modelo, versão do sistema operacional) quando o app trava. Esses dados são usados exclusivamente para identificar e corrigir bugs.

Você pode ativar ou desativar os relatórios de erros a qualquer momento em Configurações > Privacidade e Dados.

## 5. Firebase Performance Monitoring

Quando o consentimento de analytics está ativado, o Firebase Performance Monitoring pode coletar tempos de inicialização do app e métricas de requisições de rede. Esses dados são anônimos e usados para melhorar o desempenho do app.

## 6. Armazenamento Local de Dados

Todos os dados do jogo (blocos, itens, receitas, mobs, etc.), seus favoritos, listas de tarefas, notas, waypoints e configurações são armazenados **localmente no seu dispositivo** usando Room database e DataStore. Esses dados nunca são transmitidos para nenhum servidor.

Você pode excluir todos os dados de usuário armazenados localmente a qualquer momento em Configurações > Privacidade e Dados > Excluir Todos os Meus Dados.

## 7. Sincronização de Dados

O Spyglass verifica periodicamente um repositório público do GitHub para arquivos de dados atualizados. Essas requisições são requisições HTTP GET anônimas e não transmitem nenhuma informação pessoal ou do dispositivo.

## 8. Suas Escolhas

- **Diálogo de Consentimento:** Na primeira execução, você pode aceitar tudo, recusar tudo ou personalizar quais funcionalidades de coleta de dados são habilitadas.
- **Configurações:** Ative ou desative Analytics, Relatórios de Erros e Anúncios Personalizados a qualquer momento.
- **Excluir Dados:** Remova todos os dados locais do usuário (favoritos, tarefas, notas, waypoints, progresso de progressos) nas Configurações.
- **ID de Publicidade:** Você pode redefinir ou desativar o ID de publicidade do seu dispositivo nas configurações do Android.

## 9. Privacidade de Menores

O Spyglass não é direcionado a menores de 13 anos. Não coletamos intencionalmente informações pessoais de crianças. O app solicita anúncios não personalizados por padrão, que é a opção segura de privacidade conforme a COPPA.

## 10. Retenção de Dados

O Spyglass não opera nenhum servidor e não retém nenhum dado de usuário. Os dados coletados por serviços de terceiros (AdMob, Firebase) estão sujeitos às [políticas de retenção de dados do Google](https://policies.google.com/privacy).

## 11. Alterações nesta Política

Podemos atualizar esta política periodicamente. As alterações serão publicadas nesta página com uma data de "Última atualização" revisada. O uso continuado do app após as alterações constitui aceitação da política atualizada.

## 12. Contato

Se você tem dúvidas sobre esta política de privacidade, abra uma issue no nosso [repositório do GitHub](https://github.com/beryndil/Spyglass/issues).
